本项目旨在通过 Hook 注入的方式，解决特定输入法配置下，中英文模式切换混乱、状态丢失导致的输入痛点。

本项目是作者基于个人使用习惯的定制化，功能如下：
1.  **输入焦点自动重置为美式键盘：**
    * 切换到任何新的文本输入框、或焦点切换时，不再需要手动切换回英文输入法。本项目确保焦点切换后，输入模式总是自动回到 **英语 (美国) - 美式键盘**。

2.  **强制微软拼音初始为中文模式：**
    * 任何时候从美式键盘切换到微软拼音，本项目确保微软拼音总是处于 **中文输入模式**。
    * 但是并不影响使用Shift切换中英文输入的功能,只是在切换语言布局时转换为中文

## ⚙️ 前置要求与兼容性
1.  **键盘布局：** 系统中必须且只配置以下两种键盘布局：
    * **简体中文 (中国大陆) - 微软拼音**
    * **英语 (美国) - 美式键盘**

2.  **操作系统：** Windows 11。
    * **兼容性：** 目前已在 Windows 11 **24H2 版本**上测试通过，目前已使用两年，微软拼音被hook的函数未出现过变动。

## 🚀 如何使用
1.  **获取文件：** 下载最新 [Release](链接到您的Release页面) 中的压缩包解压出来的 `cheapInject.exe` 和 `mymspinyin.dll` 文件。
2.  **放置文件：** 将这两个文件放置在一个固定的、您权限足够的目录下（例如 `C:\ime123\` ）。
3.  **启动注入：** 运行 `cheapInject.exe`（运行后会弹出一个黑框瞬间消失，可以手动测试是否运行成功）。
4.  **开机自启：**
    * 为 `cheapInject.exe` 创建一个*快捷方式*。
    * 将该快捷方式放入系统的 *“启动”* 文件夹中，以确保每次开机后自动运行。
    * *提示：打开“启动”文件夹，可以在运行（Win + R）中输入 `shell:startup`。*
    * 可以直接在文件夹上面输入 启动 按回车。

* **核心 Hook 库：** 使用了 [MinHook](https://github.com/TsudaKageyu/minhook) 作为主要的 Hook 框架，感谢原作者。

* **参考** 本项目参考了 [mbbill/no_english_mode](https://github.com/mbbill/no_english_mode) 项目的设计思路。
